# MTracker 💰

Aplikasi pencatat keuangan sederhana untuk memantau pemasukan dan pengeluaran, dengan tampilan responsif.

## Fitur

- Catat transaksi (pemasukan/pengeluaran)
- Edit & hapus transaksi
- Ringkasan total saldo, pemasukan, dan pengeluaran otomatis
- Grafik visualisasi data (Chart.js)
- Data tersimpan di browser (localStorage)

## Teknologi

HTML, Tailwind CSS, JavaScript (ES Modules), Chart.js, Tabler Icons

## Cara Menjalankan

Karena project ini pakai ES Modules, harus dijalankan lewat local server (tidak bisa dibuka langsung dari file).

### pakai Python

1. cd src
2. python -m http.server

### atau pakai Node.js

1. cd src
2. npx serve
